MacBook

Keyboard Shortcuts

Keyboard Shortcuts
Shortcut Name Description
CMD + C COPY Copy selected to clipboard
CMD + V PASTE Paste clipboard contents
CMD + X CUT Cut selected and copy to clipboard
CMD + A SELECT ALL Selects All Contents of Page
CMD + Z UNDO Undo previous command
CMD + B BOLD Bold
CMD + I ITALICIZE Italic
CMD + U UNDERLINE Underline
CMD + F FIND Search the page
CMD + O OPEN Open selected file or choose file to open
CMD + K + U COMMENT Comments or uncomments selected line/lines

HTML

HTML Tags

HTML Tags
Tag Name Description
<html> HTML Wrapper Tag for all HTML files
<!--...--> Comment Used to insert comments in source code - Not displayed in browsers
<head> Head Wrapper tag that contains the metadata for the HTML file
<body> Body Wrapper tag that contains actual content of HTML file
<section> Section Defines a section in an HTML document

HTML Tables

Purpose:

Allows users to create data tables.

Table Tags
Tag Name Description
<table> Table Wrapper Tag for all HTML tables
<caption> Caption Caption/Title for HTML tables
<thead> Table Head Set of rows for headers in the table - Can only use <th> - 1/table
<tbody> Table Body Set of rows containing data for the table
<tfoot> Table Foot Set of rows that make up footer of table - 1/table
<tr> Table Row Creates a new row in the table
<th> Table Header Creates a new header cell in the table - Use scope to specify row/col
<td> Table Data Creates a new data cell in the table
Table Attributes
Attribute Name Description
colspan Column Span Specifies how many columns a <th>/<td>; element spans
rowspan Column Span Specifies how many rows a <th>/<td> element spans
scope Scope Specifies whether <th> element is a header for the column or row
